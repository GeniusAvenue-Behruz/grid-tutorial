State & Props in React

State and props are fundamental concepts in React. State is used to manage data that changes over time, while props are used to pass data from one component to another.

State Example

      
        import React, { useState } from 'react';

        const Counter = () => {
          const [count, setCount] = useState(0);

          return (
            <div>
              <p>Count: {count}</p>
              <button onClick={() => setCount(count + 1)}>Increment</button>
            </div>
          );
        };

        export default Counter;

Props Example

      
        import React from 'react';

        const Greeting = ({ name }) => {
          return <h1>Hello, {name}!</h1>;
        };

        export default Greeting;
