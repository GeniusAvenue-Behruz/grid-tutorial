Styling in React

Styling in React can be done using various methods. One popular way is to use Tailwind CSS. To get started, install Tailwind CSS:

      
        npm install tailwindcss
        npx tailwindcss init

Configure Tailwind CSS in your project and then you can use it in your components:

      
        import React from 'react';

        const StyledComponent = () => {
          return (
            <div className="p-4 bg-blue-500 text-white">
              <h1 className="text-2xl">Hello, styled with Tailwind CSS!</h1>
            </div>
          );
        };

        export default StyledComponent;
