Understanding Components in React

Components are the building blocks of a React application. A component can be a function or a class. Here is an example of a functional component:

      
        import React from 'react';

        const MyComponent = () => {
          return (
            <div>
              <h1>Hello, World!</h1>
            </div>
          );
        };

        export default MyComponent;

Components can also accept props, which are inputs to the component:

      
        import React from 'react';

        const Greeting = ({ name }) => {
          return <h1>Hello, {name}!</h1>;
        };

        export default Greeting;
